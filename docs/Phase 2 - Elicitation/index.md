This phase focuses on transforming the Phase 1 Vision and Scope into structured, traceable, and prioritized user requirements for the Sillah: Family Preventive Health Platform.

Multiple elicitation techniques were applied to ensure comprehensive requirement coverage, including interviews, surveys, document analysis, observation, and competitive review.

All extracted requirements are documented using a consistent format and prioritized using the MoSCoW method. Each requirement is linked to supporting evidence to ensure traceability and transparency.

This phase includes:

- Elicitation Plan
- Stakeholder Analysis
- Interview Notes
- Survey Findings
- Prioritized User Requirements
- Document and Competitive Analysis
