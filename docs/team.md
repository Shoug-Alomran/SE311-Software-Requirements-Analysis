## Project Background

The *Sillah (صلة)* project originated in **SE201** and was expanded in
**SE311** using the complete Requirements Engineering lifecycle:

**Inception → Elicitation → Analysis → Specification → Validation**

The documentation website was developed using **MkDocs (Material Theme)**
to ensure clarity, traceability, and academic consistency.
